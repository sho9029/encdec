# encdec v2.0

XORベースのファイル暗号化・復号ツールです。

## 特徴
- **XOR ストリーム暗号**: シンプルな XOR 演算に基づきつつ、ビット回転を組み合わせた独自のキーストリーム生成を採用。
- **バイナリ透過性**: テキスト、画像、実行ファイルなど、あらゆるファイル形式を破損させることなく処理可能。
- **完全性検証**: 復号時にパスワードの不一致やデータの改ざんを自動的に検知します。
- **後方互換**: 過去のバージョン (v1.3) で暗号化したファイルもシームレスに復号。
- **セキュアな設計**: システム依存の関数を排除し、入出力パスの厳密な検証を行っています。

## 使い方
ビルド済みの `encdec.exe` を使用します。

### 暗号化
```bash
encdec.exe -i <input_file> -o <output_file> -k <password>
```

### 復号
```bash
encdec.exe -i <encrypted_file> -o <output_file> -k <password>
```

引数なしで実行すると対話モードになります。
